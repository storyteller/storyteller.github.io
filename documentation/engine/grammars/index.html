Grammars Edit on GitHub

Grammars are the elements of your specification language. Grammars come in five basic flavors:

  1. Sentences -- a single line
  2. Tables
  3. Paragraphs -- a composite "macro" of other grammars
  4. Set Verifications
  5. Embedded Sections
